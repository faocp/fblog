+++
title = "Absolute vs relative paths"
front = "An absolute path starts from the filesystem root; a relative path starts from your current directory."
category = "navigation"
difficulty = "beginner"
weight = 60
+++

Examples:

```sh
/Users/felipe/projects/site   # absolute
../site                       # relative
```

Real-world example: scripts often use relative paths for project files, but absolute paths for fixed system locations.
