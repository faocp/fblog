Example:

```py
user = {"name": "Felipe", "admin": False}
print(user["name"])
```

Since Python 3.7, dictionaries preserve insertion order, so iterating over a dict gives keys in the order they were added.

Real-world example: dictionaries are commonly used for JSON-like data, configuration, counters, and looking up values by meaningful names.
