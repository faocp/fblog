front = "A merge conflict happens when Git cannot automatically combine competing edits."

+++

Git marks the conflicted file with conflict markers. You edit the file to choose the correct final version, then stage and commit the resolution.

Real-world example: two people edit the same line in `README.md`. Git asks a human to decide which wording should survive.
