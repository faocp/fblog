Useful options:

```sh
ls -la
ls -lh
```

`-a` includes hidden files, `-l` shows details, and `-h` makes file sizes human-readable.

Real-world example: `ls -la` helps you see config files like `.gitignore`, `.env`, or `.bashrc`.
